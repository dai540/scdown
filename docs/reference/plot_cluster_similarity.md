# Plot similarity between annotations

Plot similarity between annotations

## Usage

``` r
plot_cluster_similarity(obj, top_n_features = 200L, outdir = NULL)
```

## Arguments

- obj:

  A `scdown_obj`.

- top_n_features:

  Number of variable genes used for similarity.

- outdir:

  Optional output directory.

## Value

A list with similarity tables and plot paths.

# Recommended Pipeline Around `scdown`

Recommended Pipeline Around `scdown`

## Usage

``` r
recommended_pipeline(
  scenario = c("generic", "pbmc", "tumor_immune", "patient_comparison")
)
```

## Arguments

- scenario:

  One of `"generic"`, `"pbmc"`, `"tumor_immune"`, or
  `"patient_comparison"`.

## Value

A data frame describing suggested stages, goals, tools, and where
`scdown` fits.
